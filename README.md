# OscarVault

An immersive archive for Academy Award winners. Explore every film, director, and actor that has taken home the golden statuette since 1929.

## Features

- **Browse Winners**: Search and filter through decades of Oscar history.
- **Timeline Explorer**: Navigate the evolution of cinema through the years.
- **Oscar Battle**: Compare two award-winning films head-to-head.
- **Most Awarded**: Discover the films that have dominated the Academy Awards.
- **Modern UI**: A beautiful, cinematic interface built with React, Tailwind CSS, and Framer Motion.

## Tech Stack

- **Framework**: React 18
- **Styling**: Tailwind CSS
- **Icons**: Lucide React
- **Animations**: Framer Motion
- **Routing**: React Router 6
- **Build Tool**: Vite
